# Progettazione di Software Medicali — Patient Management System

> *Progettazione di Software Medicali / Tecnologie per la Telemedicina — Master's Degree in Biomedical Engineering, Politecnico di Torino, A.A. 2020/21*

## Project

Full UML-based design of a **patient management software system** for healthcare clinics, developed following formal medical software engineering methodology. The project covers the complete design lifecycle: from process modeling and requirement analysis, through UML specification, to interface design and testing documentation.

As a final deliverable, the team built an **interactive PowerPoint simulation** demonstrating the system's functionality through a concrete clinical scenario — a General Practice (MMG) clinic workflow.

## Deliverables

### UML Diagrams

| Diagram | File | Description |
|---------|------|-------------|
| Synopsis | `synopsis (FINALE).drawio` | High-level system overview |
| Use Case | `use case diagram_DEF.drawio` | Actors and functional requirements |
| Activity | `activity diagram_DEF.drawio` | Clinical process flows |
| Communication | `communication diagram_DEF.drawio` | Object interactions and message flows |
| Swim Lanes | `SL 1-10 (FINALE)-3.drawio` | UI design for 10 system screens |
| Workflows | `WF 1-10 (FINALE)-2.drawio` | Detailed workflows for 10 system processes |
| Interface Specs | `interfacce_DEF.drawio` | Full interface specifications |

> All `.drawio` files can be opened with [draw.io](https://app.diagrams.net/) (free, browser-based).

### Interactive Simulation

`presentations/simulazione MMG PSM.pptx` — An interactive prototype simulating the system in a GP clinic scenario, covering patient registration, appointment management, visit documentation, prescriptions, and specialist referrals.
